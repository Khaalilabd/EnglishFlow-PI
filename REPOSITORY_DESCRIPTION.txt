## Comment Ajouter la Description:
1. Allez sur https://github.com/Khaalilabd/EnglishFlow-PI
2. Cliquez sur l'icône ⚙️ à côté de "About" (en haut à droite de la page)
3. Dans le champ "Description", collez la description ci-dessus
4. Si vous avez un site web déployé, ajoutez l'URL dans "Website"
5. Cochez "Use your GitHub Pages website" si applicable
6. Cliquez sur "Save changes"
